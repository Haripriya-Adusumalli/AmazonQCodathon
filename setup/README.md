# Setup Files

This folder contains all setup and installation files for the Product Opportunity Recommendation System.

## Files

- **SETUP.md** - Comprehensive setup guide with detailed instructions
- **QUICKSTART.md** - 5-minute quick start guide
- **setup.bat** - Automated Windows setup script
- **setup.sh** - Automated Linux/macOS setup script  
- **verify-setup.py** - Setup verification script

## Usage

### Quick Setup
- Windows: Run `setup.bat`
- Linux/macOS: Run `./setup.sh`

### Verification
Run `python verify-setup.py` to check your setup.

### Manual Setup
Follow instructions in SETUP.md for detailed manual configuration.
